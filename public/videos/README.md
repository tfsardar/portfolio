Put your project demo videos here:

  project-1.mp4
  project-2.mp4
  project-3.mp4

These filenames match the entries in `lib/content.ts` — rename the file,
or update the `video` field in `lib/content.ts`, whichever is easier.

Tips:
- Keep each clip under ~15-20MB (screen-record at 1280x720, trim to
  15-30 seconds) so the page stays fast.
- MP4 (H.264) has the widest browser support.
- Optional: add a poster image (a still frame) to /public/images and
  reference it in the project's `poster` field for a nicer thumbnail
  before the video loads.
